網址縮短與本地對應管理工具

💡 使用說明：
  1. 在「短代號」欄位輸入好記的英文或縮寫（例如：wc 或 blog）。
  2. 在「原始長網址」欄位貼上要對應的完整網址（系統會自動補全 https://）。
  3. 點擊「新增對應關係」即可儲存到瀏覽器本地，隨時點擊訪問或刪除。
  4. 快速跳轉功能：在網址列後面加上 #短代號（例如 .../index.html#wc），即可自動跳轉到對應網址！
  5. 支援隨時點擊頁面最下方按鈕將此工具下載至本地離線使用。
